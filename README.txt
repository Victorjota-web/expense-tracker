# 💸 Expense Tracker

Welcome! This is a full-stack app to help you track your expenses with ease.  
From adding new expenses to viewing your balance — it's all here! 🚀

## ✨ Features
- 📥 Add & remove expenses
- 📊 See your total balance
- 🧾 Filter and manage transactions
- 🌐 Full backend integration with MongoDB

## 🛠️ Technologies
- ⚛️ React
- 🟢 Node.js + Express
- 🍃 MongoDB
- 🎨 CSS

## 🧪 How to Run Locally

1. Clone this repo
2. Install dependencies in both folders:  
   - `npm install` in `/client`  
   - `npm install` in `/server`
3. Start backend:  
   `npm run dev` (from `/server`)
4. Start frontend:  
   `npm start` (from `/client`)
5. Make sure `.env` is correctly set with:
REACT_APP_API_URL=http://localhost:5000
